# UE 灯光参数手册(为 Gemini 准备的"公开旋钮说明书")

> 背景:实评"光太硬/黑墙/色温没用" —— 此前只设了强度+衰减+RGB 颜色,**源半径/色温/源长度从未接**。
> 本文档两个用途:① 开发者备忘;② 其中【AI 旋钮】小节的措辞可直接嵌入 Gemini 提示词,
> 告诉它每个参数控制什么视觉效果、真实世界的取值锚点。

## PointLight / SpotLight 关键参数(逐个的"作用与区别")

### 强度 Intensity + 单位 Intensity Units
- 单位选 **Lumens(光通量)** 才是真实世界数值,可直接用灯具常识:床头灯 ~300lm、
  落地灯 600–900lm、吸顶灯 ~1200–2500lm、氛围/搁架小灯 ~150lm、蜡烛 ~12lm。
- Candela 是方向性强度、Unitless 是旧版任意值 —— 都不可与真实灯具对表,不用。

### 色温 Use Temperature + Temperature(K)
- **勾上 Use Temperature 后,Temperature 用黑体辐射给光"物理正确的暖/冷"**,
  比手填 RGB 自然得多(RGB 容易调出物理上不存在的脏色)。
- 锚点:火柴/篝火 1700K、蜡烛 1850K、调暗的白炽灯 2400K、**白炽灯泡 2700K**、
  卤素 3000K、暖白 LED 3000–3500K、办公荧光 4000K、日光管 5000K、**正午日光 6500K**、
  阴影/蓝天 7000–8000K。
- 与 Light Color 是**相乘**关系:用色温时 Color 留白 (1,1,1);只有霓虹/RGB 灯带这种
  **本身就是彩色的光源**才用 RGB 着色(此时不用色温)。

### 衰减半径 Attenuation Radius —— ⚠ 本次黑墙的元凶
- 这**不是**亮度衰减曲线(衰减天然是平方反比),而是影响范围的**硬截断**:
  超出半径的像素一点光都收不到。528cm 的半径放在 5.5×8.2m 的房间里 → 远墙物理上
  收不到光 → 纯黑墙。
- 性质上是引擎优化参数而非"观感"参数 → **代码定,不交 AI**:室内 = 房间对角线
  (保证任何角落都在范围内);室外点光 ≥ 2000cm。代价:半径越大开销越高,但单房间几盏灯无所谓。

### 源半径 Source Radius —— ⚠ 本次"光太硬"的元凶
- 物理含义:**发光体的实际尺寸**。0 = 数学点光源 → 本影/半影宽度为零 → 刀切硬影。
  半径越大,半影越宽、阴影越柔,镜面高光也越大(物理一致)。
- 真实锚点:裸灯泡 2–5cm、带布/纸灯罩 10–25cm、纸灯笼 20–40cm、扩散面板/吸顶 30–60cm。
- 这是 AI 能从照片直接判断的物理量(看灯具的发光面多大)→ **交 AI**。

### 软源半径 Soft Source Radius
- 在 Source Radius 之上**只**追加阴影柔化(不改高光),是廉价的"再柔一点"旋钮。
- 约定:代码取 = Source Radius(再柔化一档),不单独暴露给 AI。

### 源长度 Source Length
- 把点光拉成**胶囊/管状**:灯管、灯带、长条吊灯用。灯泡类 = 0。
- AI 看得出"这是管还是泡" → **交 AI**(灯管 30–120cm)。

### 投射阴影 Cast Shadows
- 每盏开了影的灯都画一套阴影 → 多盏同开 = 多重影子(实评"重影")+ 开销。
- 室内规则(已实装):**最亮者 = 唯一主光投影**,其余实用光不投影。代码定。

### 间接光照强度 Indirect Lighting Intensity
- 该灯对 Lumen GI 反弹的贡献倍率。默认 1.0 = 物理;调大是补黑角的"作弊"手段,
  优先用正确的衰减半径+真实流明解决,不动它。

### 体积散射强度 Volumetric Scattering Intensity
- 该灯在体积雾里的"光柱/光晕"强度。室内有雾/光束需求时才动,默认 1.0。

### 移动性 Mobility
- 我们全用 Movable(动态光,Lumen 全动态管线),无需烘焙。

## 分工表(谁定什么)

| 参数 | 决策者 | 依据 |
|---|---|---|
| 流明 | **AI** | 照片里灯具类型 → 真实灯具流明 |
| 色温 K / 彩色 RGB | **AI** | 照片光色(暖黄=白炽 2700K…);彩色仅霓虹/灯带 |
| 源半径 | **AI** | 照片里发光体的物理尺寸 |
| 源长度 | **AI** | 管状/点状 |
| 辉光球有无 | **AI** | 光源体在照片中是否可见(且有灯具锚) |
| 衰减半径 | 代码 | 房间对角线(覆盖空间,优化参数非观感) |
| 软源半径 | 代码 | = 源半径 |
| 投影层级 | 代码 | 最亮者独占投影 |
| 单位/移动性 | 代码 | Lumens / Movable 固定 |
